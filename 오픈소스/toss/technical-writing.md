## technical-writing

공식 깃허브 -> https://github.com/toss/technical-writing

1. 레포지토리를 Fork하고 Fork한 내 레포지토리를 clone한다.

2. cd technical-writing 으로 디렉토리 수정

3. 필수 구성 요소를 설치

```bash
yarn install
```

4. 로컬 서버 열기

```bash
yarn dev
```
